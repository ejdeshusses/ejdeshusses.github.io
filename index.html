Multidisciplinary

Simulation For Resource-Allocation Integer Programming Model, SWAP

  • Designed and wrote a market simulation for the SWAP project, with inputs from an interdisciplinary team.
  • Developed Design of Experiment (DOE) tables for varying different factor levels.
  • Using probability distributions of various factors, the simulation randomly generated input data, including nonprofit resources offered in the market round and bids placed on resources by other nonprofits.
  • Allocated resources were determined by solving the integer programming model using the open-source CBC solver.
  • Different nonprofit behaviors and market successes/outcomes could be observed.
  • A full explanation of the Simulated Data Experiments and the factors can be found in Section 6 of our ACM publication.

    • Tools: Python (matplotlib, pulp, numpy, pandas), DOE, ANOVA, Integer Programming
